# Frontend Components

This directory contains all React components organized by type.

## Structure

```
components/
├── ui/           # Shadcn/ui components (auto-generated)
├── features/     # Feature-specific components
└── shared/       # Shared/common components
```

## Usage

### UI Components (Shadcn/ui)

Pre-built, accessible components from Shadcn/ui. Do not modify directly.

### Features

Feature-specific components like:

- `auth/` - Authentication forms, login, register
- `forum/` - Post cards, comment sections
- `profile/` - User profile components

### Shared

Reusable components across features:

- Layout components
- Navigation
- Wrappers
- Common UI patterns
